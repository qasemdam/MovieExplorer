# 🎬 MovieExplorer

MovieExplorer is an Android application developed using Java and Android Studio. The app connects to The Movie Database (TMDB) API to display popular movies, search for movies, and view detailed information in a clean Material Design interface.

## Features

- Browse popular movies from TMDB API
- Search movies by title
- View detailed movie information
- RecyclerView with List and Grid layouts
- TabLayout with ViewPager2
- Fragments for navigation
- ProgressBar while loading data
- Notifications for success and error events
- Modern Material Design UI

## Technologies

- Java
- Android Studio
- TMDB API
- RecyclerView
- ViewPager2
- TabLayout
- Fragments
- AsyncTask
- OkHttp
- Glide
- View Binding
- Material Design

## Project Structure

- MainActivity
- MovieFragment
